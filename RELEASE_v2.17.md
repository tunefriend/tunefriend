## TuneFriend v2.17

Fixes Android back button actually working.

### Changes

- **Native back handler** — system back is intercepted in Android code (v2.16 relied on a JS listener that never connected)
- **Back on Songs / Albums tabs** — ← appears top-left; returns to Home
- **Settings** — ← returns to your previous tab
- **Home** — system back sends app to background instead of closing

### Install

Download **TuneFriend-v2.17.apk** below and install over v2.16.
