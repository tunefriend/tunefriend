## TuneFriend v2.36

### Identify song (AcoustID)
- **Identify** on Home (Android): listen ~8s → Chromaprint fingerprint → AcoustID lookup
- Match against your synced library and **Play from library** when found
- Free AcoustID API key in **Settings → Identify** (or Worker secret `ACOUSTID_CLIENT` on tunefriend.org)
- Open, MusicBrainz-linked identification — not a proprietary Shazam-style API
