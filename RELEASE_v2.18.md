## TuneFriend v2.18

Fix library sync for large collections (10k+ songs).

### Changes

- **IndexedDB storage** for synced library — no longer limited by small localStorage quota
- Clearer error message (Wi‑Fi was never the issue)
- Existing cached libraries migrate automatically on first open

### Install

Download **TuneFriend-v2.18.apk** below. After install, **Settings → Sync Library** should work for large servers.
