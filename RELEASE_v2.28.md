## TuneFriend v2.28

### Favorites playlist helpers
- **Genres → Favorite my mix** — one tap adds Rock + Alt Rock + Country + 1980s + 1990s to Favorites (capped/diversified so huge libraries don’t fill storage with one artist)
- **Favorite All** on each genre/decade screen
- Favorites **Shuffle** uses artist-spread sampling so big catalogs don’t stick on the same few artists

Then: **Favorites → Shuffle** for your custom pool instead of Shuffle All on the whole server.
