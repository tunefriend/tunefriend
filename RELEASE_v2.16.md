## TuneFriend v2.16

Android-friendly back navigation — no more accidentally closing the app.

### Changes

- **Back button** appears at the top when viewing an album, artist, or search
- **Settings** has a back button (returns to your previous tab)
- **Now Playing** uses a back arrow instead of X
- **Android system back** navigates within the app; from Home it sends the app to the background instead of closing

### Install

Download **TuneFriend-v2.16.apk** below and install over your current version.
